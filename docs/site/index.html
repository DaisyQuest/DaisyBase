Accessibility Commitments

  • Keyboard navigation starts with a skip link and visible focus state.
  • Color is never the only carrier of meaning.
  • Link labels describe destinations directly.
  • Typography, contrast, and spacing are tuned for long-form technical reading.
  • Motion is minimal and disabled further when reduced motion is requested.
