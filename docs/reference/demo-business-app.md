# Demo Business Application

## Purpose

The demo business application shows DaisyBase in a realistic enterprise-shaped stack:

- TomEE backend
- DaisyBase JDBC driver
- browser frontend
- installable demo runtime

## Backend

The backend exposes JAX-RS resources for dashboard, directory, and orders.

## Frontend

The frontend provides an operations cockpit for:

- metrics
- alerts
- employees
- customers
- products
- orders
- fulfillment actions

## Deployment Paths

- direct TomEE deployment of the WAR
- installer-driven TomEE deployment through the demo-business installer profile
